> A Vue.js project

##初始化vue项目
vue init webpack vue2-shop
##初始化express-generator
**全局安装的npm包**
```
npm list -g --depth=0 //全局安装的npm包
npm i -g express-generator //express-generator生成项目骨架
express --ejs server //项目名称server，指定使用ejs模板
cd server
npm install //安装依赖包
npm start //启动项目
```
